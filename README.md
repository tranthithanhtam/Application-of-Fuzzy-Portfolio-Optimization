# Application-of-Fuzzy-Portfolio-Optimization
Developed a multi-objective portfolio optimization model under uncertainty using fuzzy logic, optimizing Sharpe Ratio and Value at Risk Ratio. Compared NSGA-II, NSGA-III, MOEA/D, AGEMOEA-II, SMS-EMOA with a proposed ε-constraint SLSQP method, achieving significantly faster convergence on Vietnamese stock market data (2023–2025).
